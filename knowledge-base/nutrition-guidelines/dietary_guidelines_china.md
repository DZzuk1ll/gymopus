# 中国居民膳食指南（2022版）核心推荐

来源：中国营养学会《中国居民膳食指南（2022）》

## 八条核心准则

### 准则一：食物多样，合理搭配
- 每天的膳食应包括谷薯类、蔬菜水果、畜禽鱼蛋奶和大豆坚果等食物
- 平均每天摄入12种以上食物，每周25种以上
- 每天谷类食物200~300g，其中全谷物和杂豆类50~150g，薯类50~100g

### 准则二：吃动平衡，健康体重
- 维持健康体重，BMI维持在18.5~23.9 kg/m²
- 各年龄段人群都应天天进行身体活动，保持健康体重
- 食不过量，保持能量平衡
- 每周至少进行5天中等强度身体活动，累计150分钟以上
- 鼓励适当进行高强度有氧运动，加强抗阻运动
- 减少久坐时间，每小时起来动一动

### 准则三：多吃蔬果、奶类、全谷、大豆
- 蔬菜：每天摄入不少于300g，深色蔬菜应占1/2
- 水果：每天摄入200~350g新鲜水果，果汁不能代替鲜果
- 奶制品：每天摄入300~500g液态奶或等量奶制品
- 大豆及其制品：每天15~25g大豆（相当于豆腐75~150g）
- 全谷物：每天50~150g

### 准则四：适量吃鱼、禽、蛋、瘦肉
- 每周吃鱼2次或300~500g
- 每天吃畜禽肉40~75g
- 每天吃蛋类40~50g（约1个鸡蛋）
- 优先选择鱼，少吃肥肉、烟熏和腌制肉制品
- 动物性食物每天总量120~200g

### 准则五：少盐少油，控糖限酒
- 食盐：成人每天不超过5g
- 烹调油：成人每天25~30g
- 添加糖：每天不超过50g，最好控制在25g以下
- 酒：不劝酒不酗酒，儿童青少年、孕妇、乳母不应饮酒

### 准则六：规律进餐，足量饮水
- 合理安排一日三餐，定时定量，不漏餐，不暴饮暴食
- 早餐的能量占全天总能量的25%~30%
- 午餐占30%~40%
- 晚餐占30%~35%
- 每天足量饮水，少量多次
- 成年男性每天饮水1700ml，女性1500ml
- 不喝或少喝含糖饮料
- 不用饮料代替白水

### 准则七：会烹会选，会看标签
- 在生命的各个阶段都应做好准备，提高营养素养
- 认识食物，选择新鲜的营养素密度高的食物
- 学会阅读食品标签，合理选择预包装食品
- 学习传统烹饪技术，享受食物的天然美味

### 准则八：公筷分餐，杜绝浪费
- 选择新鲜卫生的食物，不食用野生动物
- 食物制备生熟分开，熟食二次加热要热透
- 讲究卫生，从分餐公筷做起
- 珍惜食物，按需备餐，提倡分餐不浪费

## 中国居民平衡膳食宝塔（2022）

每日推荐摄入量（成人，以2000kcal能量水平为参考）：

| 食物类别 | 每日推荐量 |
|---------|-----------|
| 谷薯类 | 200~300g（全谷物和杂豆50~150g，薯类50~100g） |
| 蔬菜类 | 300~500g（深色蔬菜占1/2） |
| 水果类 | 200~350g |
| 畜禽鱼蛋类 | 120~200g（水产品40~75g，蛋类40~50g） |
| 奶及奶制品 | 300~500g |
| 大豆及坚果 | 25~35g |
| 烹调油 | 25~30g |
| 食盐 | <5g |
| 饮水 | 1500~1700ml |

## 特殊人群营养建议

### 运动人群
- 运动前2-3小时适量进食碳水化合物为主的食物
- 运动后30分钟内补充碳水化合物和蛋白质
- 高强度运动后碳水:蛋白质比例建议3:1至4:1
- 运动中注意补水，每15-20分钟补充150-200ml水
- 超过1小时的运动可考虑含电解质的运动饮料
